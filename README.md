# SQL-Advanced-Practice-Set
Bu çalışma, SQL'in ileri seviye konuları olan tetikleyiciler (triggers), saklı yordamlar (stored procedures) ve karmaşık fonksiyonlar üzerindeki yetkinliğimi pekiştirmek için çözdüğüm senaryoları içermektedir. Sınav hazırlık sürecimde, veritabanı mantığını en ince ayrıntısına kadar kavramak adına oluşturduğum kapsamlı bir antrenman setidir.
